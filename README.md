# Assignment 1 - Personal Portfolio

This repository contains a simple, responsive personal portfolio web application built as part of the Assignment 1 requirements. It showcases my projects and skills using HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: The layout adapts seamlessly to desktop, tablet, and mobile devices.
- **Dark/Light Mode**: User preference for theme is saved in local storage.
- **Dynamic Greeting**: The greeting message changes based on the time of day.
- **Semantic HTML**: Code is structured using semantic tags for better accessibility and SEO.
- **Interactive Elements**: Includes smooth scrolling and form handling.

## Setup Instructions

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/FarisKFUPM/202277660-Faris-assignment1.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd 202277660-Faris-assignment1
    ```
3.  **Open `index.html` in your browser:**
    - You can simply double-click the file to open it.
    - Alternatively, use a live server extension (like Live Server in VS Code) for a better development experience.

## Deployment

The project is static and can be deployed easily to platforms like GitHub Pages, Netlify, or Vercel.

**GitHub Pages:**
1.  Go to the repository settings on GitHub.
2.  Navigate to the "Pages" section.
3.  Select the `main` branch as the source and save.
